# payloadcms-ogreniyorum
Bu repo çeşitli dil modelleri tarafından üretilmiş payloadcms v3 ile uyumlu müfredat listesini içerir.
Müfredat listesine "/CurriculumByAI" klasörü altında dil modellerine göre ayrılmış dosya içeriklerinden erişebilirsiniz.
23 Haziran 2026 tarihi itibari ile adım adım tüm dil modelleri tarafından üretilmiş içerik listesi aktarılacaktır. 
ModuleByDeepseek-v4-pro/Module-1 klasöründe ilk ders notları incelenebilir.
PayloadCMS-V3'ü öğrendikçe ders notlarını aktarmaya devam edeceğim.
